# Shopify UX Resources — E-Commerce Research for Storefront Audits

A curated collection of e-commerce UX research we rely on when conducting [Shopify storefront UX audits](https://sprucepixel.com/pages/storefront-audit). Every recommendation in a Spruce Pixel audit is grounded in published research, not opinion.

Maintained by [Spruce Pixel](https://sprucepixel.com) — Shopify storefront UX audits by former Shopify employees.

## Baymard Institute

Baymard maintains the largest research-based resource on e-commerce UX, with thousands of usability guidelines drawn from large-scale testing.

- [Homepage & Category UX](https://baymard.com/blog) — How users browse and filter products on e-commerce sites
- [Checkout Usability](https://baymard.com/checkout-usability) — Benchmark studies on checkout UX across major e-commerce storefronts
- [Mobile E-Commerce UX](https://baymard.com/blog/mobile-commerce-design) — Mobile shopping behavior and design patterns specific to small screens
- [Product Page UX](https://baymard.com/product-page) — Image galleries, descriptions, reviews, and purchase decision factors
- [E-Commerce Search UX](https://baymard.com/ecommerce-search) — Site search behavior, autocomplete, and results page design

## Nielsen Norman Group (NNGroup)

NNGroup publishes foundational usability research used across industries, with strong coverage of e-commerce and mobile.

- [E-Commerce UX Research](https://www.nngroup.com/topic/e-commerce/) — Trust signals, product comparison, and purchase flows
- [Mobile UX](https://www.nngroup.com/articles/mobile-ux-study-guide/) — Touch targets, responsive layouts, and mobile navigation
- [Navigation Design](https://www.nngroup.com/topic/navigation/) — Information architecture, menus, and wayfinding
- [Visual Design Principles](https://www.nngroup.com/topic/visual-design/) — Visual hierarchy, typography, and layout

## Google

Google's developer resources cover performance, accessibility, and Core Web Vitals — all of which directly affect both UX and search ranking for Shopify stores.

- [Core Web Vitals](https://web.dev/vitals/) — Performance metrics that impact user experience and SEO
- [PageSpeed Insights](https://pagespeed.web.dev/) — Real-world page performance measurement for mobile and desktop
- [Web.dev UX Collection](https://web.dev/learn/) — Accessibility, responsive design, and performance optimization guides

## Recommended Reading

- _Don't Make Me Think_ by Steve Krug — The classic on web usability and intuitive navigation
- _Laws of UX_ by Jon Yablonski — Design principles backed by psychological research
- _The Design of Everyday Things_ by Don Norman — Foundational thinking on human interaction with designed objects
